remove_red_eyeOperable Checkers

Perceivable Checkers will try to evaluate whether a given website is easy to operate

2.1 Keyboard Accessible

 All functionality of the content is operable through a keyboard interface without requiring specific timings for individual keystrokes, except where the underlying function requires input that depends on the path of the user's movement and not just the endpoints.

2.2 Enough Time

 copy the url of the webpage to the following textbox and click "TEST" button, the result will be the score of text alternatives in the given text. The lower the score, the less the text alternative.

2.3 Seizures

 Do not design content in a way that is known to cause seizures.

2.4 Navigable

 Provide ways to help users navigate, find content, and determine where they are.
